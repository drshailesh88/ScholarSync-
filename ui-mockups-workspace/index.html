Design Direction Summary

Key shifts from the current SaaS-style design to workspace-style:

  • Navigation: Heavy 264px sidebar → 64px icon rail (like Google Workspace apps)
  • Colors: Dark slate + neon glow → Warm stone canvas (#FAFAF9) with per-app identity colors
  • Typography: Plus Jakarta Sans → Inter (UI) + Source Serif 4 (content). Lighter weights, more whitespace.
  • Surfaces: Glass morphism + blur → Clean white cards, subtle 1px borders, no blur
  • Copy: "Luminous studio... flow state... glow" → "Think clearly. Write deeply." Calm, functional.
  • Spacing: Dense, packed layout → Generous whitespace. Content breathes.
  • Feeling: SaaS dashboard → Researcher's quiet desk. Deep work, not feature overload.
